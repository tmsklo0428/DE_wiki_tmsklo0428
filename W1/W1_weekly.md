[잘한 점]
이번 주에는 새로운 Python 함수와 SQL 문법을 직접 실행하며 결과를 확인하고, 모르는 내용은 바로 찾아보는 방식으로 학습했다. 
덕분에 과제와 직접 관련 없는 예제도 꼼꼼히 살펴보며 새로운 문법과 활용 방법까지 익힐 수 있었다.
또한 데이터를 시각화할 때 그래프가 보기 불편하다고 느껴 가독성을 높이도록 개선한 점이 잘한 부분이라고 생각한다. 


[아쉬운 점]
함수 주석을 작성하지 않는 등 코드 품질을 충분히 고려하지 못했다. 
앞으로는 기능 구현에만 집중하기보다 코드의 가독성과 유지보수성도 함께 고려하며 작성하는 습관을 기르도록 노력해야겠다.
또한 GitHub에 파일을 업로드하는 과정에서 폴더 위치를 잘못 설정해 파일을 다시 이동하는 작업이 필요했다. 
앞으로는 커밋이나 푸시 전에 프로젝트 구조와 파일 위치를 한 번 더 확인하여 같은 실수를 반복하지 않도록 해야겠다.
